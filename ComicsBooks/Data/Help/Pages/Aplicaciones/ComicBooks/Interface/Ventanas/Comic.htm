Ventana de visualización de cómics

La ventana de visualización de cómics e imágenes es la que nos permite leer los cómics o ver nuestras imágenes.

Esta ventana se divide en tres secciones:

  • En la parte superior se encuentra la barra de herramientas para visualización de cómics.
  • En la parte izquierda vemos una serie de thumbnails con las páginas de nuestro cómic ( o en el caso de un directorio los archivos de imagen ) .
  • En la parte central se dispone la imagen.

Para pasar las páginas podemos utilizar la rueda central del ratón ( una vez que lleguemos a la parte inferior de la imagen se pasará a la siguiente o a la anterior cuando veamos la parte superior ) o el combo de páginas de la barra de herramientas o pulsando directamente sobre su thumbnail en la parte izquierda o bien pulsando el botón secundario del ratón sobre la imagen para ver un menú de opciones y pulsando sobre la opción adecuada.

Para ajustar el nivel de zoom podemos utilizar también la rueda central del ratón mientras mantenemos pulsada la tecla de mayúsculas.

Como dijimos anteriormente si pulsamos el botón secundario del ratón aparecerá un menú de opciones que reproduce en parte las opciones de la barra de herramientas desde la que podemos cambiar el zoom, pasar de página, rotar la imagen de la página actual o mostrar una lupa para ver con mayor detalle la imagen:

Para que desaparezca la lupa simplemente debemos volver a pulsar sobre la imagen.

La aplicación recuerda los ajustes de zoom y rotación que hayamos hecho sobre una imagen para las siguientes imágenes de forma que las veamos todas con el mismo nivel de zoom o rotadas para facilitar la lectura.

Nota: esta misma ventana se utiliza en la creación de cómics. Para más información diríjase a la página creación de cómics .
