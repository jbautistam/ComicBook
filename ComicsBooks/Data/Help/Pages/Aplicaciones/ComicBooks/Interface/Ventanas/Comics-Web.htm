Ventana de visualización de cómics Web

Una vez seleccionado un cómic en la ventana de lista de cómics Web se abre esta ventana para visualizarlos:

Desde aquí simplemente pulsando en las flechas de atrás y delante o modificando la fecha en la barra de herramientas se descargará el archivo y se visualizará en la pantalla.

Las imágenes se descargan en su ordenador de forma que no es necesario volver a descargarlas, puede acceder de esa forma a las imágenes una vez grabadas desde su propio ordenador sin conexión a Internet.
