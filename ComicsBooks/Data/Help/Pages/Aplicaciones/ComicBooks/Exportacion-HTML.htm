Exportación de cómics e imágenes a formato HTML

Una vez tenemos abierta una imagen, un directorio o un cómic en la ventana de visualización de cómics podemos exportar las imágenes a formato HTML.

Para ello simplemente debemos ir a la opción Archivo | Exportar | HTML del menú principal .

Una vez pulsada la opción del menú nos aparecerá la siguiente ventana:

En esta ventana debemos seleccionar:

  • El directorio donde deseamos dejar el archivo HTML, las imágenes y los thumbnails ( o miniaturas ) .
  • El título de la página ( inicialmente el mismo que el archivo que estemos visualizando ) .
  • El ancho con el que se grabarán los thumbnails de las imágenes.
  • El número de columnas de imágenes en la tabla.
  • El ancho máximo que tendrán las imágenes grabadas ( si se deja el valor 0 las imágenes se grabarán con el mismo tamaño que el original ) .
  • Si deseamos o no ver el resultado directamente en el explorador.

Una vez seleccionados los valores adecuados podremos pulsar el botón Aceptar para generar las páginas:

La aplicación copiará en el directorio seleccionado las imágenes del archivo de cómic y los archivos de miniaturas necesarios. Además crea una página llamada Gallery.htm que permite visualizar en un explorador la galería de imágenes:
