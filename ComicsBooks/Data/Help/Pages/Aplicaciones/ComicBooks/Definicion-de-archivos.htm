¿Qué es un archivo de cómic?

Un archivo de cómic es un archivo especial que contiene una serie de imágenes para su lectura utilizando aplicaciones especiales.

Estos archivos suelen ser archivos comprimidos con utilidades como WinZip o WinRar a las que se les cambia la extensión a CBZ en el caso de archivos ZIP o CBR en el caso de archivos RAR ( o con menos frecuencia con la extensión CBT para archivo TAR ) .

Este tipo de archivos resulta muy útil a la hora de transmitir los cómics por Internet dado que no es necesario almacenar una larga lista de imágenes si no que se sustituye por un único archivo.

Para leer este tipo de archivos se puede modificar la extensión a .ZIP o .RAR según sea el caso, descomprimir el archivo y posteriormente ver las imágenes. En cualquier caso resulta mucho más sencillo utilizar aplicaciones como ComicBooks que además ofrecen posibilidades adicionales como recordar la última página leída, cambiar de página utilizando el ratón, diversos niveles de zoom, etcétera.
