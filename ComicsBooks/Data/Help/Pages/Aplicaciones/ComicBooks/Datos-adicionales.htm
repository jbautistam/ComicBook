Definición de datos adicionales de un cómic

Para permitir la clasificación de cómics la aplicación ComicBook añade la posibilidad de introducir datos adicionales en un archivo de cómic.

Para ello una vez tenemos abierto un cómic en la ventana de visualización de cómics podemos pulsar el botón marcado con una I en la barra de herramientas

En este formulario podemos introducir datos como el título, el autor o dibujante, la serie a la que pertenece el cómic, etc... que luego se utilizarán en la ventana de colecciones para clasificar los cómics.

Los datos se almacenan dentro del propio cómic, por tanto para guardar las modificaciones debemos grabar el archivo como se indica en la sección de creación de cómics .
