Cómics Web

Las viñetas de algunos periódicos o algunos cómics como Calvin / Hobbes o Dilbert se encuentran accesibles diariamente desde determinadas páginas Web.

La aplicación ComicBooks permite descargar y visualizar las viñetas de un día determinado para verlas cómodamente en su ordenador.

Para ello en la ventana de lista de cómics Web se muestra una lista de todas los cómics Web que puede leer la aplicación.

Si pulsa dos veces sobre una de ellas se abrirá la ventana de visualización de cómics Web desde donde podrá descargar y visualizar los cómics.
